# ♟️ Jogo de Xadrez em C# (Console)

Um jogo de xadrez simples desenvolvido em **C#**, como uma aplicação de console no **Visual Studio**. Ideal para estudos e experimentação com lógica de programação, estruturas de dados e regras básicas do xadrez.

---

## 🎮 Funcionalidades

- ✅ Jogo para **2 jogadores locais**
- ♟️ Movimentação válida para todas as peças
- 🚫 Detecção de **movimentos inválidos**
- 👑 Suporte a **xeque** e **xeque-mate**
- 🧩 Exibição do **tabuleiro no terminal**
- 🔁 Reinício de partida via terminal

---

## 🛠️ Tecnologias Utilizadas

- Linguagem: **C#**
- Plataforma: **.NET Console Application**
- IDE: **Visual Studio** (Windows)
- Ambiente: **Terminal de Console (Prompt interno do VS)**

---

## 📌 Observações

- Este projeto **não possui interface gráfica (GUI)** — todo o jogo é controlado via terminal.
- As peças são representadas com **letras** (ex: `K` para rei, `Q` para rainha).
- As jogadas são digitadas no formato tradicional, como `e2 para e4`.
